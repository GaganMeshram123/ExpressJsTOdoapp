to setup the project 

1. clone the project
'''
git clone https://github.com/GaganMeshram123/ExpressJsTOdoapp.git
'''

2. move in the directory where project is closed and coded 

'''
cd ExpressJsTOdoapp
'''

3. install dependencies 

'''
npm install
'''
 
4. turn up the server
'''
node index.js
'''
